# Conscious Campus Registration Form

A registration form for Conscious Campus that collects data and stores it in Google Sheets.

## Files
- `index.html` - The main registration form
- `form.css` - Styles for the form
- `Code.gs` - Google Apps Script code for handling form submissions

## Setup
1. The form is connected to a Google Sheet for data storage
2. Form submissions are handled by Google Apps Script
3. Data is stored in real-time in the connected Google Sheet
